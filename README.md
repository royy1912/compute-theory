# Roy's channel

### 前言
因為我個人很喜歡吃拉麵，所以我想把台南好吃的拉麵店推廣給大家，因此做了這個line bot

## 使用方法
點開聊天室後需要隨便說一句話，之後機器人便會問想要吃的拉麵種類，回答完後機器人就會傳出該種類推薦的拉麵店和google地圖
